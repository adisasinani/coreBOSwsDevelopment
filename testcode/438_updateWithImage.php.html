Update Record with Image

Purpose:

Permit updating records with one or more associated image attachments.

This call is exactly the same as Update, in fact it actually uses Update, the only difference is that there is a special field parameter that permits you to send in the images for each image field.

Send as:

POST

Parameters:

attachmentsarray:
  fieldname => array
    'name'=>filename
    'type'=>file mime type
    'size'=>file size
    'content'=>base64_encode({file})
  {repeat for all image fields}
