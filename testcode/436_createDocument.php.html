Create Document with Attachment

Purpose:

Permit the creation of new Documents records with an associated attachment.

This call is exactly the same as Create, in fact it actually uses Create, the only difference is that there is a special field parameter that permits you to send in the attachment.

Send as:

POST

Parameters:

filenamearray:
  'name'=>basename({file}), // no slash nor paths in the name
  'size'=>filesize({file}),
  'type'=>{mime type of file},
  'content'=>base64_encode({file})

Additional Information:

REST Document Manipulation Enhancements
