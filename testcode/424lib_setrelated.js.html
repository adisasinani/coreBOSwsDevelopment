SetRelated

Purpose:

Establish many to many (m:m) relations between records.

Profile:

SetRelated(relate_this_id:Id,with_this_ids:string/array):boolean

Send as:

POST

Parameters:

relate_this_idstring ID of record we want to related other records with
with_this_idsstring/array either a string with one unique ID or an array of IDs to relate to the first parameter

Response:

true if all went well, error otherwise

URL Format:

http://corebos_url/webservice.php?operation=SetRelated&sessionName=[session id]&relate_this_id=[id main record]&with_this_ids=[json encoded list of records]

Additional Information:
