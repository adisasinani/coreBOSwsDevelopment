Retrieve Attachment

Purpose:

Retrieve file attached to a document as a base64 encoded string. This method was added instead of modifying the already existing Retrieve method to avoid having to download all attachments when only a list is required and so we wouldn't have to modify the existing interface of the retrieve method.

Profile:

retrievedocattachment(id:Id,returnfile:boolean):Map

Send as:

POST

Parameters:

idcomma separated list of document identifiers to obtain
returnfiletrue if we need to really get the file, false to just get the information of the attachment

Response:

A map object with all the information of the attachment

['result'][$docId]['filetype']
['result'][$docId]['filename']
['result'][$docId]['filesize']
['result'][$docId]['attachment'] (base64 encoded)

URL Format:

http://corebos_url/webservice.php?operation=retrievedocattachment&sessionName=[session id]&id=[id documento]&returnfile=[0|1]

Additional Information:

REST Document Manipulation Enhancements
